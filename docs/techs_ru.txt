Использовались бибилиотеки  и фреймворки:
    - PySide6 == 6.7.2 – вся работа приложения сделана с использованием его
    - openai-whisper – для распознавания голоса
    - Torch – фреймворк для создания нейросетей
    - Pillow and numpy использовались для обработки изображений
    - easyocr – распознаёт текст на изображениях
    - PyMuPDF – для распознавания информации из PDF файла
    - pdf2docx – конвертирует из PDF в .docx документ
Дизайн написан на CSS.
